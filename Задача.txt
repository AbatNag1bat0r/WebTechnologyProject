1. сделать клонирование репозитории в терминале через команду git clone https://github.com/AbatNag1bat0r/WebTechnologyProject.git
2. установить необходимые зависимости: 1) скачать node.js если не установлен, проверить через команду 
node -v; 2) устновить зависимость npm через комнаду npm install (но надо находиться в директории приложения react(в вашем случае название директории front))
3) инструмент vite через команду npm install -g create-vite;
3. вы запускаете ДЖанго через комнаду python manage.py runserver и в другом терминале запускаете реакт отдельно через команду npm run dev


после того как у всех студентов будет одна версия, можете рабоать в своей ветке. когда каждый студент закончит свою работу ему надо отправить свою версию на гитхаб команда git push origin (название ветки); 
вы можете на гитхабе проверить была ли импортирована ваша версия. если есть дополнительная мейн ветка можете сделать pull request и merge

остальыне студенты могу обновить на своем локальном компьютере версию которую вы отправили через команду git pull origin (дополнительная мейн ветка)
